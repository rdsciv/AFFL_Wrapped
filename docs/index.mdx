AFFL Wrapped is an open reference implementation for cinematic fantasy football history. It transforms twelve years of validated ESPN records and six seasons of advanced Wrapped analytics into shareable annuals without requiring a database or private API at runtime.

## Design principles

1. **Stories before tables.** Every metric must help explain a season.
2. **Definitions before rankings.** Power and luck are useful only when reproducible.
3. **Static delivery.** Public history should be fast, durable, and cheap to host.
4. **Honest provenance.** Missing source coverage is labeled rather than inferred away.
5. **League personality.** The product should feel like a living record, not enterprise BI.

## What ships publicly

The repository includes generated presentation datasets, UI source, documentation, and test contracts. It does not include ESPN authentication values, raw API payloads, or the full historical database.
