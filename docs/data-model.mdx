## Public story contract

Each season object contains:

- champion, runner-up, and title game;
- league pulse and activity totals;
- all-play power standings and luck leaders;
- four matchup awards;
- season and single-game player impact;
- draft and undrafted spotlights where supported;
- one summary card per team;
- explicit coverage metadata.

## Provenance limits

ESPN does not return complete player-level draft and event-level transaction history before 2018. Older annuals still include validated standings, scores, lineups, acquisition counters, and team outcomes. The interface labels the narrower source coverage instead of presenting inferred events as facts.

One 2016 season aggregate differs from ESPN's internal display by nine points; the source snapshots and normalized calculations are retained and documented in the private validation archive.
