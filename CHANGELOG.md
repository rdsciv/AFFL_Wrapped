# Changelog

All notable project changes are documented here.

## 1.0.0 - 2026-07-14

### Added

- Twelve AFFL season annuals covering 2014 through 2025.
- Championship, power, luck, matchup, player, draft, waiver, and team story modules.
- Static GitHub Pages deployment with permanent annual URLs.
- Source-backed data coverage notes and pre-2018 ESPN caveats.
- Mintlify-ready documentation, community health files, and contribution workflows.
- Branded repository artwork and reproducible animated demo.

### Changed

- Replaced the original full-stack starter structure with a focused static Next.js project.
